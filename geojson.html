What is GeoJson

GeoJSON is a geospatial data interchange format based on JavaScript Object Notation (JSON). It defines several types of JSON objects and the manner in which they are combined to represent data about geographic features, their properties, and their spatial extents. GeoJSON uses a geographic coordinate reference system, World Geodetic System 1984, and units of decimal degrees [1] . The examples that you will use are based on points (latitude, longitude). The page at [2] explains GeoJSON at an introductory and comprehensive level. You can see many examples at: http://geojson.xyz/
